🚀 E-Commerce Automation Framework

An enterprise-level automation framework built using Robot Framework for UI, API, and Database testing with CI/CD and Docker integration.

✨ Highlights
UI Automation using Selenium
API Testing using RequestsLibrary
Database Validation
Parallel Execution with Pabot
CI/CD using GitHub Actions
Dockerized Test Execution
Reusable Page Object Model
HTML Reports & Artifact Management


🛠️ Tech Stack

Robot Framework • Selenium • Python • RequestsLibrary • DatabaseLibrary • Docker • GitHub Actions • Pabot
